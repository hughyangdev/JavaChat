## 📁 JavaChat
> Java Network 구현 테스트  

채팅 기능 구현

	100: 클라이언트 입장 정보 (100|아이디|대화명)
	300: 이모티콘 메시지 (300|이모티콘 번호)
	400: 일반 대화 메시지 (400|글자색|메시지)
	500: 귓속말 (500|귓속말 보내는사람 대화명|귓속말 메시지)
	600: 대화명 변경 (600|기본대화명|새로변경할대화명)
	700: 중복 대화명이 있을 경우 (700|)
	800: 퇴장(로그아웃) (800|아이디)
	900: 채팅종료 (900|아이디)
    
  

### 📑 환경
- JDK 1.8
- Tools : Eclipse, NetBeans
